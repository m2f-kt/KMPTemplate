    **Step 1: Create RBAC plugin**
    Create `server/auth/src/main/kotlin/com/m2f/server/auth/authorization/RoleAuthorization.kt`:
    - Define `class RoleConfig { var roles: Set<String> = emptySet() }`
    - Create route-scoped plugin using `createRouteScopedPlugin`:
      ```kotlin
      val RoleAuthorizationPlugin = createRouteScopedPlugin(
          name = "RoleAuthorizationPlugin",
          createConfiguration = ::RoleConfig
      ) {
          val requiredRoles = pluginConfig.roles
          on(AuthenticationChecked) { call ->
              val principal = call.principal<JWTPrincipal>()
              val userRole = principal?.payload?.getClaim("role")?.asString()
              if (userRole == null || userRole !in requiredRoles) {
                  call.respond(
                      HttpStatusCode.Forbidden,
                      ErrorResponse(code = "USER_FORBIDDEN", message = "Insufficient permissions")
                  )
              }
          }
      }
      ```
    - Add convenience extension:
      ```kotlin
      fun Route.withRole(vararg roles: String, build: Route.() -> Unit) {
          install(RoleAuthorizationPlugin) { this.roles = roles.toSet() }
          build()
      }
      ```
    - Import `AuthenticationChecked` from `io.ktor.server.auth`
    - Import `ErrorResponse` from `com.m2f.template.models.dto`

    **Step 2: Create UserService**
    Create `server/auth/src/main/kotlin/com/m2f/server/auth/service/UserService.kt`:
    - Constructor takes `UserRepository`
    - `context(raise: Raise<DomainError>) suspend fun getProfile(userId: String): UserResponse`:
      1. Parse UUID from userId
      2. Find user by id via `userRepository.findById(uuid)` -- if null, `raise(UserNotFound())`
      3. Map to `UserResponse(id, email, name, role)`

    - `context(raise: Raise<DomainError>) suspend fun updateProfile(userId: String, request: UpdateProfileRequest): UserResponse`:
      1. Parse UUID from userId
      2. If both name and email are null, raise a validation error (nothing to update)
      3. Validate provided fields using `zipOrAccumulate` accumulated validation:
         - If `request.name != null`, validate with `validateName(request.name)`
         - If `request.email != null`, validate with `validateEmail(request.email)`
         - Use `raise.withError(...)` pattern to convert `NonEmptyList<FieldError>` to `IncorrectInput`
      4. If email changed, check uniqueness: `userRepository.findByEmail(validEmail)` must be null or same user
      5. Call `userRepository.updateProfile(uuid, validName, validEmail)`
      6. Fetch and return updated profile

    - `context(raise: Raise<DomainError>) suspend fun getUserById(targetUserId: String): UserResponse`:
      1. Parse UUID from targetUserId
      2. Find user by id -- if null, `raise(UserNotFound())`
      3. Map to UserResponse
  </action>

    - `fun Route.userRoutes(userService: UserService)` extension function
    - All routes wrapped in `authenticate { ... }` since they require JWT
    - `route("/api/users") { ... }` block:

    ```kotlin
    authenticate {
        route("/api/users") {
            // Get own profile
            get("/me") {
                conduitAuth { userId ->
                    userService.getProfile(userId)
                }
            }

            // Update own profile
            put("/me") {
                conduitAuth { userId ->
                    val request: UpdateProfileRequest = getModel()
                    userService.updateProfile(userId, request)
                }
            }

            // Admin: get any user by ID
            withRole("ADMIN") {
                get("/{id}") {
                    conduitAuth { _ ->
                        val targetId = getStringParam("id")
                        userService.getUserById(targetId)
                    }
                }
            }
        }
    }
    ```
