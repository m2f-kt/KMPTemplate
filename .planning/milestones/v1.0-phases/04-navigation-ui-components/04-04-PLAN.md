**TerminalAlert.kt:**
```kotlin
enum class AlertVariant { Info, Success, Warning, Error }

@Composable
fun TerminalAlert(
    message: String,
    modifier: Modifier = Modifier,
    variant: AlertVariant = AlertVariant.Info,
    title: String? = null,
    onDismiss: (() -> Unit)? = null,
)
```
- Row or Column layout with variant-colored left border (like accent card) or full background
- Variant color mapping: Info -> (infoBg, info), Success -> (successBg, success), Warning -> (warningBg, warning), Error -> (errorBg, error)
- Border from TerminalTheme.borders, radius from TerminalTheme.radius
- Title in typography.sm with fontWeight.SemiBold, message in typography.sm
- Optional dismiss "X" using a clickable BasicText or icon

**TerminalBadge.kt:**
```kotlin
enum class BadgeVariant { Default, Accent, Success, Warning, Error }

@Composable
fun TerminalBadge(
    text: String,
    modifier: Modifier = Modifier,
    variant: BadgeVariant = BadgeVariant.Default,
)
```
- Small inline pill shape: `RoundedCornerShape(TerminalTheme.radius.pill)`
- Variant color mapping: Default -> (surface, text, border), Accent -> (accentMuted, accent), Success -> (successBg, success), Warning -> (warningBg, warning), Error -> (errorBg, error)
- Small text (typography.xs), tight horizontal padding
- No click handler -- badges are display-only

**TerminalProgress.kt:**
```kotlin
@Composable
fun TerminalProgress(
    modifier: Modifier = Modifier,
    progress: Float? = null,
)
```
- When `progress` is non-null (0.0-1.0): determinate bar showing filled portion
- When `progress` is null: indeterminate animated bar (use `rememberInfiniteTransition` with `animateFloat` for a sliding indicator)
- Track: full-width rounded rect in `inset` color
- Fill: rounded rect in `accent` color, width = progress * trackWidth
- Height: ~6-8dp (verify against .pen), corner radius: radius.pill
- Use `Canvas` or `Box` with `Modifier.fillMaxWidth().height(h).clip(shape).background(trackColor)` and inner `Box` for fill

**TerminalTooltip.kt:**

@Composable
fun TerminalTooltip(

- On hover (Desktop/WASM) or long press (mobile), shows a small floating label

**TerminalCheckbox.kt:**
```kotlin

    label: String? = null,

)
```
- Use Foundation `Modifier.toggleable(value = checked, enabled = enabled, role = Role.Checkbox, onValueChange = onCheckedChange)` for accessibility
- Checkbox box: ~18x18dp square with rounded corners (radius.sm)
- Unchecked: surface background with border (borders.default width, border color)
- Checked: accent background, checkmark (unicode "\u2713" or custom Canvas drawing) in surface color
- Optional label in typography.sm, color text, separated by gap.sm
- Disabled state: reduced opacity via TerminalTheme.opacity.medium

**TerminalSwitch.kt:**
```kotlin

    modifier: Modifier = Modifier,

```
