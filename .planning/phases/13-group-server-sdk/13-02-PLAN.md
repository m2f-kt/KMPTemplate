**1. Create `GroupService.kt`** following `AuthService.kt` pattern:
- Constructor: `class GroupService(private val groupRepository: GroupRepository, private val membershipRepository: MembershipRepository, private val userRepository: UserRepository, private val authService: AuthService)`
- All methods use `context(raise: Raise<DomainError>)` for error handling

**Group CRUD methods:**
- `createGroup(request: CreateGroupRequest, userId: String): GroupResponse`
  - Parse userId to Uuid
  - Check slug uniqueness via `groupRepository.findBySlug()` -- raise `GroupAlreadyExists` if exists
  - Insert group, insert membership for creator with role "OWNER"
  - Return GroupResponse with memberCount=1

- `getGroup(groupId: String, userId: String, userRole: UserRole): GroupResponse`
  - Parse groupId to Uuid
  - Find group or raise GroupNotFound
  - If userRole is NOT PowerAdmin: verify user is member of this group via `membershipRepository.findByUserAndGroup()`, raise GroupForbidden if not a member
  - Return GroupResponse with memberCount from `membershipRepository.countByGroup()`

- `updateGroup(groupId: String, request: UpdateGroupRequest, userId: String, userRole: UserRole): GroupResponse`
  - Parse groupId, find group or raise NotFound
  - Authorization: user must be OWNER or ADMIN in the group (check membership role), OR be PowerAdmin
  - Update via repository, return updated GroupResponse

- `deleteGroup(groupId: String, userId: String, userRole: UserRole)`
  - Parse groupId, find group or raise NotFound
  - Authorization: user must be OWNER of the group, OR be PowerAdmin
  - Delete group (cascade memberships via DB or manual delete memberships first)

- `listAllGroups(): List<GroupResponse>` -- PowerAdmin only, called from route with withRole guard

**Member management methods:**
- `getMembers(groupId: String, userId: String, userRole: UserRole, cursor: String?, limit: Int): PaginatedMemberResponse`
  - Parse groupId, find group or raise NotFound
  - Authorization: user must be ADMIN/OWNER in group, OR PowerAdmin. Regular members cannot list all members.
  - Query membershipRepository with cursor pagination
  - For each membership, look up user details (email, name) from UserRepository
  - Encode cursor as last userId string, compute hasMore from result size vs limit
  - Return PaginatedMemberResponse

- `addMember(groupId: String, request: AddMemberRequest, userId: String, userRole: UserRole): MemberResponse`
  - Authorization: ADMIN/OWNER in group or PowerAdmin
  - Check user exists (UserRepository.findById), raise UserNotFound if not
  - Check not already member, raise MemberAlreadyInGroup if exists
  - Insert membership, return MemberResponse

- `removeMember(groupId: String, targetUserId: String, userId: String, userRole: UserRole)`
  - Authorization: ADMIN/OWNER in group or PowerAdmin
  - Cannot remove OWNER (raise CannotRemoveOwner)
  - Delete membership

- `registerMember(groupId: String, request: RegisterMemberRequest, userId: String, userRole: UserRole): MemberResponse`
  - Authorization: ADMIN/OWNER in group or PowerAdmin
  - Register user via AuthService.register() (create RegisterRequest from RegisterMemberRequest fields)
  - Add newly created user to group with specified role
  - Return MemberResponse

Helper method:
- `private fun requireGroupRole(userId: Uuid, groupId: Uuid, minRole: GroupRole): MembershipRecord` -- finds membership, checks role level is >= minRole, raises GroupForbidden if insufficient

**2. Create `GroupRoutes.kt`** following `AuthRoutes.kt` + `UserRoutes.kt` pattern:
```kotlin
fun Route.groupRoutes(groupService: GroupService) {
    authenticate {
        // Group CRUD -- Admin+ system role required to create
        withRole(UserRole.Admin, UserRole.PowerAdmin) {
            post<Groups.Create> {
                conduitAuth(HttpStatusCode.Created) { userId ->
                    val request = getModel<CreateGroupRequest>()
                    groupService.createGroup(request, userId)
                }
            }
        }

        // Read/Update/Delete -- group-level auth checked in service
        get<Groups.ById> { route ->
            conduitAuth { userId ->
                val role = // extract UserRole from JWT principal
                groupService.getGroup(route.groupId, userId, role)
            }
        }

        post<Groups.Update> { route ->
            conduitAuth { userId ->
                val request = getModel<UpdateGroupRequest>()
                val role = // extract UserRole from JWT
                groupService.updateGroup(route.groupId, request, userId, role)
            }
        }

        post<Groups.Delete> { route ->
            conduitAuth { userId ->
                val role = // extract UserRole from JWT
                groupService.deleteGroup(route.groupId, userId, role)
                mapOf("message" to "Group deleted")
            }
        }

        // List all groups -- PowerAdmin only
        withRole(UserRole.PowerAdmin) {
            get<Groups.ListAll> {
                conduit {
                    groupService.listAllGroups()
                }
            }
        }

        // Member management
        get<Groups.Members> { route ->
            conduitAuth { userId ->
                val role = // extract UserRole from JWT
                groupService.getMembers(route.groupId, userId, role, route.cursor, route.limit)
            }
        }

        post<Groups.AddMember> { route ->
            conduitAuth { userId ->
                val request = getModel<AddMemberRequest>()
                val role = // extract UserRole from JWT
                groupService.addMember(route.groupId, request, userId, role)
            }
        }

        post<Groups.RemoveMember> { route ->
            conduitAuth { userId ->
                val role = // extract UserRole from JWT
                groupService.removeMember(route.groupId, route.userId, userId, role)
                mapOf("message" to "Member removed")
            }
        }

        post<Groups.RegisterMember> { route ->
            conduitAuth { userId ->
                val request = getModel<RegisterMemberRequest>()
                val role = // extract UserRole from JWT
                groupService.registerMember(route.groupId, request, userId, role)
            }
        }
    }
}
```

Note: To extract UserRole from JWT in conduitAuth, read the principal's role claim. Create a helper function `RoutingContext.getUserRole(): UserRole` that extracts the JWTPrincipal role claim and converts it via `UserRole.fromString()`.
