  <action>
Add the compose preview tooling dependency to the designsystem module's commonMain dependencies block. The version catalog entry already exists as `compose-ui-tooling-preview`.

In `commonMain.dependencies`, add:
```
implementation(libs.compose.ui.tooling.preview)
```

This matches how composeApp uses the same dependency in its commonMain block. The annotation `org.jetbrains.compose.ui.tooling.preview.Preview` is then available in commonMain for all targets.

Do NOT add `compose.preview` (Android-only) or `compose.uiTooling` (debug-only Android tooling). The `libs.compose.ui.tooling.preview` library from JetBrains provides the multiplatform `@Preview` annotation that works across all KMP targets.
