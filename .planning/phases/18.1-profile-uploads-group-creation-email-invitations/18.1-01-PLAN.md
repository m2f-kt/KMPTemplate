    2. UserRoutes.kt: Add PUT /api/users/me/avatar endpoint:
       ```kotlin
       put<Users.Me.Avatar> {
           // Multipart parsing OUTSIDE conduitAuth (per Phase 18 pattern)
           val multipart = call.receiveMultipart()
           var fileBytes: ByteArray? = null
           var fileName: String? = null
           var contentType: ContentType? = null
           
           multipart.forEachPart { part ->
               when (part) {
                   is PartData.FileItem -> {
                       fileName = part.originalFileName
                       contentType = part.contentType
                       fileBytes = part.provider().toByteArray()
                   }
                   else -> {}
               }
               part.dispose()
           }
           
           conduitAuth { userId, _ ->
               ensureNotNull(fileBytes) { InvalidContent }
               ensureNotNull(contentType) { InvalidContent }
               
               // Upload to S3 using FileService
               val fileResponse = fileService.upload(
                   fileBytes = fileBytes!!,
                   fileName = fileName ?: "avatar",
                   contentType = contentType!!,
                   userId = userId,
               )
               
               // Update user's avatarUrl
               userRepository.updateAvatarUrl(Uuid.parse(userId), fileResponse.url)
               
               // Return updated user profile
               val user = userRepository.findById(Uuid.parse(userId))
               ensureNotNull(user) { UserNotFound() }
               user.toUserResponse()
           }
       }
       ```
       
       Note: Requires FileService injection into UserRoutes. Update the route function signature to include fileService parameter.
