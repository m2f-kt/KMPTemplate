    3. Add `getMyMemberships` method to `GroupService.kt`:
       ```kotlin
       /**
        * Get all group memberships for the current user.
        * Returns a lightweight summary with groupId, groupName, and GroupRole.
        * No RBAC check needed — users can always see their own memberships.
        */
       suspend fun getMyMemberships(userId: String): List<MembershipSummary> {
           val uid = Uuid.parse(userId)
           val memberships = membershipRepository.findByUserId(uid)
           return memberships.map { record ->
               val group = groupRepository.findById(record.groupId)
               MembershipSummary(
                   groupId = record.groupId.toString(),
                   groupName = group?.name ?: "",
                   groupRole = GroupRole.fromString(record.role),
               )
           }
       }
       ```
       Note: No `Raise<DomainError>` context needed — this cannot fail with a domain error (empty list is valid). Import `MembershipSummary` from the DTO package.

    4. Add the route handler to `GroupRoutes.kt` inside the `authenticate` block (NOT inside any `withRole` block since all users can access their own memberships):
       ```kotlin
       // Inside authenticate { ... } block, before the withRole blocks:
       get<Users.Me.Memberships> {
           conduitAuth { userId ->
               groupService.getMyMemberships(userId)
           }
       }
       ```
       Import `Users` from routes package (already imported as `Groups` is there). Actually, this route logically belongs in `UserRoutes.kt` but `GroupService` lives in the groups module. Since `GroupRoutes.kt` already has access to `GroupService`, add it here. Add import for `Users` from routes.
