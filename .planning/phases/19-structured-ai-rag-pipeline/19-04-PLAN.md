  <name>Task 1: RagService for query embedding and retrieval</name>
  <files>
    server/ai/src/main/kotlin/com/m2f/server/ai/rag/RagService.kt

  <action>
    Create `RagService` in `server/ai/rag/`:

    Constructor parameters:
    - `embedder: LLMEmbedder` -- for embedding the query text
    - `db: R2dbcDatabase` -- for creating PgVectorStorage instances per request
    - `relevanceDetector: RelevanceDetector` -- for auto-RAG detection
    - `aiDispatcher: CoroutineDispatcher` -- for embedding calls

    Methods:

    1. `suspend fun checkAndRetrieve(query: String, groupId: Uuid, userId: Uuid, isAdmin: Boolean, topK: Int = 5): RagContext?`
       - First, check relevance: `relevanceDetector.check(query)`
       - If `needsContext == false`, return null (no RAG needed)
       - If `needsContext == true`, call `retrieve()` with the optimized searchQuery (or original query as fallback)
       - Return `RagContext(chunks: List<String>)` or null

    2. `suspend fun retrieve(query: String, groupId: Uuid, userId: Uuid, isAdmin: Boolean, topK: Int = 5): RagContext?`
       - Create a `PgVectorStorage` scoped to this groupId/userId/isAdmin
       - Embed the query: `withContext(aiDispatcher) { embedder.embed(query) }`
       - Search similar: `storage.searchSimilar(queryVector, topK, minSimilarity = 0.3)` -- 0.3 threshold filters out irrelevant noise
       - If no chunks found, return null
       - Return `RagContext(chunks = results.map { it.content })`

    3. `fun formatContext(ragContext: RagContext): String`
       - Format chunks for injection into system prompt:
         ```
         [Document Context]
         The following excerpts from uploaded documents may be relevant to the user's question.
         Use them to provide accurate, informed answers. Do not mention these documents unless the user asks about them.

         ---
         {chunk 1}
         ---
         {chunk 2}
         ---
         ```
       - Hidden context per user decision -- no citations, no mention of sources

    Data class:
    - `data class RagContext(val chunks: List<String>)`
  </action>
