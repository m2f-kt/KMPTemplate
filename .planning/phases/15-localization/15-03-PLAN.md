- ProfileMutation: Change `SetFieldErrors(val errors: Map<String, String>)` to `SetFieldErrors(val errors: Map<String, StringKey>)`, change `SetServerError(val error: String?)` to `SetServerError(val error: StringKey?)`
- ProfileViewModel.handleSaveProfile(): Replace inline validation strings with StringKey values:
  ```kotlin
  val errors = mutableMapOf<String, StringKey>()
  if (current.editName.isBlank()) {
      errors["name"] = StringKey.VALIDATION_NAME_BLANK
  }
  if (current.editEmail.isBlank()) {
      errors["email"] = StringKey.VALIDATION_EMAIL_BLANK
  } else if (!current.editEmail.contains("@")) {
      errors["email"] = StringKey.VALIDATION_EMAIL_INVALID
  }
  ```
- ProfileViewModel.handleLoadProfile() and handleSaveProfile() server error handling: Map `error.code` to StringKey:
  ```kotlin
  ifLeft = { error ->
      val key = StringKey.fromCode(error.code) ?: StringKey.GENERIC_ERROR
      sendMutation(ProfileMutation.SetServerError(key))
  }
  ```
