## Decisions Made
- Per-request `AIAgent` instead of singleton `AIAgentService` -- required for injecting per-request streaming callbacks into the strategy graph
- Removed `SayToUser`, `AskUser`, `ExitTool` tools entirely -- with streaming, the LLM's plain text output IS the response delivered via SSE, making conversation tools unnecessary
- Used `AIAgentGraphStrategy<String, Any>` output type -- matches the reference implementation pattern; actual content is delivered via the `processFrame` callback, not the strategy return value
- Updated system prompt to remove "Always use the respond tool" instruction -- plain text responses are now the intended mechanism
