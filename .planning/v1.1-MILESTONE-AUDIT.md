## Tech Debt Register

### Phase 13: Group Server & SDK (5 items)

1. **Missing integration test — registerMember endpoint**
   Severity: Low. Production code works (AdminPanel UI exercises it). Test gap only.

2. **Missing integration test — RBAC edge case: member cannot add members**
   Severity: Low. RBAC is enforced server-side. Untested permutation.

3. **Missing integration test — RBAC edge case: admin can manage but not delete group**
   Severity: Low. Delete restricted to Owner role. Untested permutation.

4. **Missing integration test — pagination boundary (empty next page)**
   Severity: Low. Pagination logic is standard cursor-based. Edge case untested.

5. **Anti-pattern: joinedAt="" in addMember/registerMember response**
   Severity: Low. Database stores correct timestamp. Immediate response returns empty string. Re-fetching returns correct value. Cosmetic issue.

### Phase 15: Localization (1 item)

6. **WASM locale in memory only**
   Severity: Low. Documented limitation. `window.__customLocale` + `navigator.languages` monkey-patch works but page reload needed for full Compose Resources locale switch. All other targets persist correctly.

### Phase 10: MVI ViewModel Foundation (1 item)

7. **SharingStarted.Lazily was a UAT-discovered fix**
   Severity: Info. Phase 10 verification documented `WhileSubscribed(5_000)` as correct. UAT revealed back-navigation caused `scan()` to reset from `initialState`. Changed to `Lazily` in commit `3ff3bba`. Not a current issue — just a documentation/verification accuracy note.

**Total: 7 items across 3 phases. No blockers.**
